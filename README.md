# Telaffuz Ustası 🎙️

Almanca ve Rusça telaffuz çalışma uygulaması.

## Kurulum

```bash
npm install
npm run dev
```

## Netlify Deploy

`netlify.toml` dosyası zaten ayarlı. GitHub'a push ettikten sonra Netlify'da repoyu bağlayın, otomatik deploy olur.

## AI Analiz

[console.anthropic.com](https://console.anthropic.com) adresinden API anahtarı alın. Uygulama içinde "AI Analiz" butonuna basınca key girişi açılır.
